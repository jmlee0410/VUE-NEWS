# vue-news

## Project setup

```cmd
npm install
```

### Compiles and hot-reloads for development

```cmd
npm run serve
```

### Compiles and minifies for production

```cmd
npm run build
```

### Lints and fixes files

```cmd
npm run lint
```

### Vue Router 설치

```cmd
npm i vue-router --save
```

#### Vue Router Mode

- hash mode(default), history mode
- hash mode
  - URL 해시를 사용하여 전체 URL을 시뮬레이트
  - URL이 변경될때 페이지가 다시 로드되지 않음
